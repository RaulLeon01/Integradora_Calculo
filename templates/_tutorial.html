Guía rápida

Cómo usar el programa

1

Escribe la ecuación

Usa formatos como y' + 2y = 0 o y'' + 3y' + 2y = 0.

2

Agrega condiciones iniciales

Captura y(0) y, si es de segundo orden, también y'(0).

3

Resuelve

El sistema mostrará la transformada, $Y(s)$, la solución final y su gráfica.

4

Opcional: activa Modo IA

Si configuras tu API key, podrás pedir una explicación más guiada del resultado.
